The idea is try to build a SNS with 3D. The user experience is like "second life"(move/look/touch all is OK), but based on web tech. So far use the google o3d api. But may be replaced by html5. The demo include a room with some furniture, a pet, and a photo wall which show your facebook photos that retrieve the facebook open api.

1. The demo only pass testing on ff 3.6 mac ox 10.6, have to install o3d plugin

download ff3.6 for mac : http://ju.joyqom.com/o3dsns/Firefox3.6.3.dmg

download o3d plugin:  download ff3.6 for mac : http://ju.joyqom.com/o3dsns/o3d.dmg

2. This demo is based on o3d offical demo "home-desginer"(I didnot do any arts，all 3d model is from o3d offical sample)

3. User can walk in he room freely (by mouse)

4. Use a 2d hud to show Message, including message， event and notification。

5. This demo include a pet

 

5. This demo include a photowall. Photos are runtimely retrieved from your facebook accont's first album，so you will be prompt to login facebook when you enter the demo.
