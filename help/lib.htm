How to write external lib for NCSE
When you develop a module for c script, there are some limits you must notice.
  1. NC Script does not support Object, So you must export Interface but not class.
  2. Script only support 4 data type: long , short, char, float. All kinds of pointer can used as long.
  3. Script does not support reference. If you want to use an out parameter, you can pass the address of the variable.
  4. The function exported must be defined as a '__stdcall' function, if you use MSVC.
  5. You must use the def file to export identifier instead of using __declspec(dllexport), if you use MSVC.
