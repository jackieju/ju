SE_SetRegKey
File: se_interface.h
void SE_SetRegKey(char* sUsername, char* key);
Description
Set the register code
Remarks
If you not set the code, you can only call SE_ExecScript no more than 100 times. If you want to use the se unlimitedly, you should buy a register key.
Parameters
char* key
[in]the key you got from author
char* sUsername
[in]user name

void
