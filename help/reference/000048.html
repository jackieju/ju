SE_Start
File: se_interface.h
int SE_Start(char* szConfigFile);
Description
Init NC script engine
Remarks
This function will init script engine, compile all NC file defined in config file 'Source File List' group. So if these NC script call any function of external lib, you must load those lib using SE_LoadLib() before you call SE_ComileAll

Return Value
int- see return code list
