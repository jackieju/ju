SE_Stop
File: se_interface.h
int SE_Stop(long nMillionSeconds, long force);
Description
Stop Script Engine and release resouces used by SE

Parameters
long force
[in]If force = 1, when the interval elapses and there are still running scripts, all runing scripts will be terminated.

long nMillionSeconds
[in] Specifies the time-out interval, in milliseconds. The function returns if the interval elapses and not all runing script exit.
Return Value
int - please see return code list
