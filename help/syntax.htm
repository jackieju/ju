  1. Overview

    2. The NC language is a reduced C language. It's syntax is a subset of C syntax. So some features of C are not supported by NC language, which should be take a look.

  2. Syntax Specification

    3. The syntax is defined in atg file(used by cocoR).
    NOTICE: Although the switch sentence can be parsed, but it was not be implemented, so you can not use switch sentence.

  3. Data Type Supported

    4. char(1 bytes), short(2 bytes), long(4 bytes), float(4 bytes) NOTICE: unsigned types are not supported by NC.

  4. Data Type Cast

    5. NC noly support the implicit type cast, but not the explicit cast.

  5. Operator Supported

    6. Mathematic	:	+ - * / % ++ --
Logical		:	< > <= >= ==
Relational	:	&& || !
others		:	& ~

  6. Function Call

    7. There are tow kinds of function: NC Function and external lib function
    If call a NC function, the called function must be defined before the caller.
    If call a external lib function, the called function must be loaded before the
    caller be compiled.

  7. Other Specifications

    1. The variables can only be defined in the start of the function, and the
      variable can not be initialized while being difined.
    2. The following operators are not supported:
      +=, -=, *=, /=, %=, |, &
    3. class and structure are not supported in the version
    4. swith statement are not supported
    5. bit operators are not supported
    6. dose not support new and delete
    7. dose not support comma expression
