A Web2.0 website with Semantic Web featureclosed in 2009

Feature: allow user add his favorite web resource (website, video, doc ...), and share, search... with others'

Architecture:
Server side: Apache with connector to tomcat (java)
client side: IE plugin, ff plugin

Tech features:
1. use a defined format url(protocol://[uuid]/[dbid]) to identify a resource
2. use JCR repository to store resource and relationship
3. use db to index resource
4. use RDF(semantic web) to store resource property
5. version 2.0 changed to webos UI
