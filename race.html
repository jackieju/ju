3D online racing game

Based on the open source race game TORCS, which is by pure c and non-network game.
The idea is create a nice 3D game which can embed dynamic in-game-ad from the advertising affliate.

Architecture:
Server side: Written by Standard c++, include lobby and game world, use lots of fancy algorithm to make a smooth racing world on internet
client side: The race engine, based on TORCS. Plugin lots of dynamic ad.

Tech features:
1. Game cross linux and window.
2. Can render dynamic ad on the place in game, like wall, car surface, pit stop ground...
3. Has ranking system
